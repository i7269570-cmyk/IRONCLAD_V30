🔴 1. 데이터 동기화 (이거 안 하면 망함)

👉 3대 돌리면 생기는 문제:

시간 다름 / 데이터 형식 다름 / 누락 위치 다름

👉 해결:

- timestamp 기준 통일 (UTC or KST 고정)
- 동일 schema 유지
- 최종 병합 구조 필요
 
### 🔴 2. 패턴 라벨 기준 명확화

👉 지금 위험한 부분:

“이게 상승인가?”
“이게 횡보인가?”

👉 기준 없으면:

* 전략 왜곡됨
* 결과 해석 불가

👉 반드시 정의:


상승: - n봉 동안 +x% 이상

횡보: - 변동폭 y% 이하

변동성: - ATR 기준 이상


👉 숫자로 고정해야 함

## 4️⃣ 앞으로 전략 찾을 때 핵심 구조

👉 이렇게 해야 한다: 전체 데이터 ❌  패턴별 데이터 ⭕

### 예

Type A (상승) → Momentum 전략

Type B (횡보) → Mean Reversion

Type C (변동성) → Breakout
```

👉 이게 “정답 구조”

## 5️⃣ 지금 수준에서 절대 하지 말 것

❌ 데이터 합쳐서 하나로 분석
❌ 패턴 구분 없이 전략 찾기
❌ 주식 전략 그대로 복붙
```


## 6️⃣ 한 단계 위 전략 (중요)

regime_filter → 현재 시장 상태 판단 → 전략 자동 선택
